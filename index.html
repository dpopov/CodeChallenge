News of the Day

Requirements

Using the News API, create an app to be able to search the news.

The main view

The main view should: Show a list of News Sources. It's up to you which of the characteristics provided by the API you display but it must include the News Source's Icon

From the main screen the user should be able to search the news sources by name.

The second view

The second view should show the news stories from the selected new's source

Keep in mind that you will have to choose how to handle the variable availability of the "sortBysAvailable" key

You can use HTTP libraries like AlamoFire

You may use 3rd party dependencies through CocoaPods, Carthage, and Gradle in helping you build your app. If you use CocoaPods or Carthage, please don't commit 3rd party source code. Your solution should include a README explaining how to set up and run the project.

Bonus points

Bonus points for a nice and interesting UI.
